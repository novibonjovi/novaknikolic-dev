Blackjack Game

This is a website with a blackjack game playable against the computer.
The goal of the game is to beat the computer by having a card count of close to or equal to 21. The player and the computer start each round with two cards then the player can choose to either draw more cards or to keep the cards. If the player has a higher count than the computer and less than 22 he wins, otherwise the computer wins.

This is another project I did for myself to practice javascipt. I found an article online which helped me with the logic of creating and shuffling a deck of 52 cards.

The blackjack game wont work on mobile because I didnt add media querries to the css code.
